"HierarchyFilter" adds a three level deep filter engine to your WooCommerce store with each level depending on each other.

== Description ==

This WooCommerce plugin lets you easily define a three level deep hierarchical filter.
Especially useful if product display is dependant on three attributes.
For example, you can assosciate products with certain vehicles.

== Features ==

* Additional product data tab for displaying assosciations, e.g. "Suitable Vehicles".
* Shortcode support
* Widget support
* CSV-Importer
* Custom cookie name
* Definable text strings

== Installation/Usage ==

1. Extract the *.zip and upload its content to the `/wp-content/plugins/` directory - alternatively you can just upload the *.zip through Plugins >> Add New >> Upload Plugin.
2. Go to HierarchyFilter >> Add New and add a new entry.
3. Go to a product and on the upper right side, select one or multiple entries you just have defined.
4. Place the HierarchyFilter widget anywhere you wish.
5. Shortcode usage: [hierarchyfilter_dropdown]
6. Use the filter on the page/post where you have inserted HierarchyFilter and you will get redirected to your storefront with only suitable products being shown.
